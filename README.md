## Run Locally

Clone the project

```shell
  $ git clone https://github.com/vtuberwiki/wiki
```

Go to the project directory

```shell
  $ cd wiki
```

Install dependencies

```shell
  $ npm install
```

Start the server

```shell
  $ npm run dev
```
